Cumbia.JS

A humble JS framework

Cumbia aims to give you a thin layer between your HTML and JS because sometimes you just need a little bit of magic.

Progressive Enhancement

Cumbia believes in HTML. The Web is primarily content and for most cases your JS should be optional and minimal

Extensible

Easy to make it yours and add some sparkles of your own magic in case what is there is not enough

Tiny

With less than 5Kb of JS you have the power to create reusable components
